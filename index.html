    Truly Extensible Architecture

    Nimbus Cloud Transfer is built for universality, utilizing a pure, loosely coupled API design.

    PHP Native Core

    Out of the box, Nimbus runs strictly on standard PHP and SQLite. Deploy it on any cPanel or shared host immediately without SSH access.

    Node.js Plug-and-Play

    Need high-performance WebSocket workers instead of PHP daemon loops? The frontend API is agnostic. You can swap out the backend processor for a Node.js daemon without touching the UI.

    100% Open Source

    Nimbus is designed to be built upon. Extend the cloud providers, modify the authentication gateways, or customize the Bootstrap 5 DOM.
